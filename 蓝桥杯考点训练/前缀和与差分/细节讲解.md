# __前缀和与差分__
## __1. 一维与二维前缀和__
### __一维前缀和__
1. 定义:数组s[i]表示原数组A从第一个元素加到第i个元素的和
2. 预处理：s[i] = a[i - 1] + A[i]
3. 急速查询：求区间[L,R]的和，直接计算s[R] - s[L - 1]

### __二维前缀和__
1. 预处理：s[i][j]表示左上角（1，1）到右下角（i,j）的最大矩阵和。s[i][j] = s[i - 1][j] + s[i][j - 1] - s[i - 1][j - 1] + A[i][j]
2. 急速查询：求左上角（x1，y1）到右下角（x2，y2）的子矩阵和
ans = s[x2][y2] - s[x1 - 1][y2] -s[x2][y1 - 1] + s[x1 - 1][y1 - 1]

## __2.一维与二维差分__
### __一维差分__
1. 痛点：让区间[L,R]里面的每一个数都加上C
2. 定义：差分数组D[i] = A[i] - A[i - 1]（核心性质：差分数组的前缀和就是原数组）
3. 修改：只需要修改两个端点，中间的完全不用管
4. 还原：修改完所有区间后，对D数组求一边前缀和，就得到了修改后的最终数组

### __二维差分__
当题目要求你对矩阵中摸一个子块内的所有元素都加上C的时候使用
1. 修改：对左上角（x1,y1）到右下角（x2，y2）的矩形区域全部加C
2. 右下角和左上角符号一致是加，另外两个对角是减
3. 还原：对差分矩阵D求一遍二维前缀和

# __双指针__

## __1.同向双指针(滑动窗口/尺取法)__
1. 应用场景： 寻找满足特定条件的"最短/最长连续子区间"
2. 运行机制：两个指针L，R都从左端点出发，绝不回退
    · R指针主动向右扩张，负责把新的元素吞进窗口，直到窗口内的状态满足要求
    · L指针负责向右搜索，负责把旧的元素吐出窗口，用于满足要求的前提下寻找最优解
3. 伪代码：
```c++
int l = 1,r = 1,sum = 0,ans =2e9;
while(r <= n){
    sum += a[r];
    while(sum >= target){
        ans = min(ans , r - l + 1);
        sum -= a[l];
        l++;
    }
    r++;
}
```

## __2.相向双指针(对撞指针)__
1. 应用场景：已排序的数组中"两数之和"的问题，或者回文判定
2. 运行机制：左指针L从数组头部1开始，右指针R从数组尾部n开始，两人面对面向中间靠拢，直到相遇
   · 如果a[L] + a[R] > target , 说明当前和太大了，必须让R左移
   · 如果a[L] + a[R] < target , 说明当前和太小了，应该让L右移
